Windows battery report

Fri Feb 8, 2019 by @harkx in  general, sysadmin  blog, windows

Some tips are awesome, got this one from a colleague.. How to check the battery of your laptop? (health, runtime, …)

Just run this simple command (in a command prompt as administrator) and it will generate a nice html report (by default in the folder you’re in):

powercfg /batteryreport
