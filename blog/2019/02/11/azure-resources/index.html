Azure resources

Mon Feb 11, 2019 by @harkx in  azure  blog, azure

Here’s a list of my go-to Azure resources, docs, blogs and other websites. There is a lot of information on these. Be careful; Azure is constantly changing so some articles can get outdated pretty quickly.

I welcome suggestions to complete this list.

Microsoft Documentation

  • https://docs.microsoft.com/en-us/azure/ - The official documentation.
  • https://docs.microsoft.com/en-ca/learn/ - Learn Azure!
  • https://azure.microsoft.com/en-us/pricing/calculator/ - Azure pricing calculator.
  • https://azure.microsoft.com/en-us/migration/ - Azure migration center.

Blogs

  • https://azure.microsoft.com/en-us/blog/ - The Microsoft Azure blog.
  • http://www.aidanfinn.com/ - Azure and other content.
  • https://blog.azureandbeyond.com/
  • https://www.azurefieldnotes.com/ - Azure and Azure stack content.
  • https://www.azurefromthetrenches.com/
  • https://buildazure.com/ - All about Azure, great blog with weekly newsletter.
  • https://www.christiaanbrinkhoff.com/ - Virtual desktops. (RDS, WVD, ..)
  • https://www.danielstechblog.io/ - Azure content, Kubernetes, IoT.
  • https://gregorsuttie.com/ - Azure, DevOps content
  • https://www.petri.com/ - A lot of good content on Azure.
  • https://pixelrobots.co.uk/ - Azure and sysadmin stuff.
  • https://rasmusg.net/
  • https://www.thomasmaurer.ch/
  • https://thomasthornton.cloud/
  • https://www.yobyot.com/ - Azure and AWS content.

Community

  • https://askazure.io/ - Ask an Azure Architect, very informative Slack channel with users, customers and Microsoft employees.
  • https://www.reddit.com/r/AZURE/ - This and /r/sysadmin are my favorite subreddits.
  • Awesome Azure:
    • https://github.com/kristofferandreasen/awesome-azure
    • https://github.com/surgicalcoder/awesome-azure
    • https://github.com/bensadeghi/Awesome-Azure-Advanced-Analytics

Video content

  • https://channel9.msdn.com - Channel9 videos.
  • https://www.youtube.com/user/windowsazure - Azure’s home on YouTube.
  • https://www.youtube.com/user/OfficeGarageSeries/ - Microsoft Mechanics, tech show on different Microsoft topics, including Azure.
  • https://azure.microsoft.com/en-us/resources/videos/home/ - Azure videos on different topics.
    • https://azure.microsoft.com/en-us/resources/videos/azure-friday/ - Different topic every week.

Other

  • https://azurespeedtest.azurewebsites.net/
  • https://aka.ms/azurebackup_agent - Latest version of Azure Recovery Services Agent.
  • https://azureinfohub.azurewebsites.net/ - A resource of publicly available content.
  • https://azureprice.net/ - Azure VM comparison.
  • https://www.vmchooser.com/vmoptimizer
